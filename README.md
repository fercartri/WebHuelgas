# Web de gestión de app Android Huelgas360

Este proyecto es una aplicación web diseñada para la gestión de ubicaciones ("ubicaciones") de una institución, probablemente el Colegio Santa María La Real de Huelgas. Proporciona una interfaz de usuario simple y funcional para realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en el catálogo de ubicaciones.

## Tecnologías usadas

La aplicación está construida con un stack de tecnologías moderno y robusto:
- Framework: Next.js
- Lenguaje: TypeScript
- Estilos: Tailwind CSS y PostCSS
- Base de Datos: Firebase Firestore
- Almacenamiento de Fotos: Supabase Storage

## Funcionalidad

- Login
- Visualización de ubicaciones
- Gestion (CRUD) de dichas ubicaciones
